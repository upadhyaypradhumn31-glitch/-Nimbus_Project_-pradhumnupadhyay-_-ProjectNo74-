# Employee Leave and Attendance Tracker - Project 74

## Project Objective
To develop a comprehensive C program that automates employee attendance tracking, leave management, and salary deduction calculations. The system helps organizations efficiently manage HR operations by providing automated tracking and reporting capabilities.

## Features
- Employee Management (Add, View, Search employees)
- Attendance Tracking and Leave Management
- Automatic Salary Deduction Calculations
- File Handling for Data Persistence
- Comprehensive Report Generation
- Input Validation System
- User-friendly Menu Interface
